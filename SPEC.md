# 抖音直播间弹幕读取与自动回复系统

## 1. 项目概述

### 项目名称
**DouyinDanmu** - 抖音直播间弹幕读取与智能回复系统

### 核心功能
这是一个桌面应用程序，可以实时读取指定抖音直播间的弹幕内容，并根据预设规则或AI智能分析自动回复观众消息。

### 目标用户
- 抖音主播（需要与观众互动）
- 直播运营团队
- 自动化直播管理需求者

## 3. 功能规范

### 核心功能

#### 3.1 抖音直播间连接
- **输入**: 抖音直播间的 room_id 或分享链接
- **方式**: 通过抖音开放API或WebSocket连接弹幕服务器
- **认证**: 支持扫码登录或Cookie认证
- **状态反馈**: 实时显示连接状态

#### 3.2 实时弹幕捕获
- **数据获取**: 监听直播间WebSocket消息
- **解析内容**:
  - 用户昵称
  - 弹幕文本
  - 发送时间
  - 用户ID
  - 粉丝等级（如果有）
- **显示**: 实时滚动显示在弹幕列表
- **存储**: 可选保存到本地文件

#### 3.3 自动回复系统
- **规则匹配模式**:
  - 关键词匹配（如：问价、发货）
  - 正则表达式匹配
  - 完全匹配
- **回复模板**:
  - 支持多个预设回复
  - 随机选择回复
  - 间隔控制（避免刷屏）
- **智能回复**:
  - 基于关键词的预设回复库
  - 可自定义规则
- **发送方式**: 通过API发送弹幕到直播间

#### 3.4 回复规则管理
- **添加规则**: 关键词 + 回复内容
- **编辑规则**: 修改现有规则
- **删除规则**: 移除不需要的规则
- **规则优先级**: 按添加顺序匹配
- **开关控制**: 启用/禁用单个规则

#### 3.5 历史记录
- **弹幕历史**: 保存所有捕获的弹幕
- **回复历史**: 保存所有发送的回复
- **导出功能**: 导出为JSON或CSV格式
- **清空功能**: 清除历史记录

### 用户交互流程

#### 连接直播间
1. 用户输入直播间链接或room_id
2. 点击"连接"按钮
3. 系统验证并建立连接
4. 显示连接成功/失败提示
5. 开始接收弹幕

#### 配置自动回复
1. 在规则管理区域点击"添加规则"
2. 输入触发关键词
3. 输入回复内容
4. 保存规则
5. 规则自动生效

#### 查看和操作
1. 弹幕实时滚动显示
2. 点击弹幕可查看详情
3. 可手动回复特定弹幕
4. 查看统计信息

#### 弹幕消息
```python
{
    "id": "唯一标识",

    "timestamp": "发送时间",
    "fan_level": 0,

    "replied": False

```

#### 回复规则
```python
{
    "id": "唯一标识",
    "keyword": "触发关键词",
    "match_type": "exact|contain|regex",
    "replies": ["回复1", "回复2"],

    "created_at": "创建时间"

```

### 边界情况处理
- **网络断开**: 自动重连机制，最多重试5次
- **无效链接**: 提示用户检查输入
- **API限流**: 实现请求间隔控制
- **弹幕过快**: 限制显示速度，可暂停滚动
- **认证失效**: 提示重新登录

## 4. 技术实现

### 技术栈
- **GUI框架**: PyQt6
- **HTTP请求**: requests
- **WebSocket**: websocket-client
- **数据存储**: SQLite (本地数据库)
- **并发处理**: asyncio + threading
