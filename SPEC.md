# 抖音直播间弹幕读取与自动回复系统 (CLI版)

## 1. 项目概述

### 项目名称
**DouyinDanmu-CLI** - 抖音直播间弹幕读取与智能回复命令行工具

### 核心功能
这是一个命令行应用程序，可以实时读取指定抖音直播间的弹幕内容，并根据预设规则自动回复观众消息。

### 目标用户
- 抖音主播（需要与观众互动）
- 直播运营团队
- 喜欢命令行操作的用户

---

## 2. 功能规范

### 核心功能

#### 2.1 直播间连接
- 支持输入直播间链接或房间号
- 自动解析各种格式的直播间URL
- WebSocket实时连接
- 断线自动重连

#### 2.2 实时弹幕捕获
- 实时显示弹幕内容（用户名+内容）
- 显示弹幕发送时间
- 记录用户进入直播间事件
- 弹幕历史保存

#### 2.3 自动回复系统
- 关键词匹配自动回复
- 支持多条回复（随机选择）
- 可开关自动回复功能
- 回复间隔控制

#### 2.4 规则管理
- 添加回复规则（关键词 + 回复内容）
- 查看规则列表
- 删除规则
- 启用/禁用规则

#### 2.5 数据管理
- 保存弹幕历史到本地
- 保存回复规则到本地
- 导出数据为JSON格式

    "time": "发送时间(HH:MM:SS)",

## 3. 技术实现

### 技术栈
- **Python**: 3.8+
- **HTTP请求**: requests
- **WebSocket**: websocket-client
- **美化输出**: rich (可选)

### 项目结构
```
/workspace/
├── main.py                 # 主程序入口
├── requirements.txt        # 依赖列表
├── SPEC.md                # 本文档
└── data/                  # 数据目录
    ├── history.json       # 弹幕历史
    └── rules.json         # 回复规则
```

### 使用方法

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

3. 输入直播间链接或房间号开始监控
